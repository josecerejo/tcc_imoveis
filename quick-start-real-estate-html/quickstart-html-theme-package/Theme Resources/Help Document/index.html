How to edit the logo and header section of the template

File : All .html files

  • Locate this block of code, in between the first <span> tag, this is going to be your website title or the title of your company
  • The second <span> tag would be your company slogan. You can safely delete this current line if you don't want to put any text or words below your website title
  • Next is go to your template package, under theme resources folder, unzip the PSDs.zip and open up quick start logo template.psd with Photoshop or any of your favourite image editor that supports .psd file type
  • Set up your logo using that psd template and then save it as logo.gif
  • Now back to your .html files, locate the images folder and overwrite the logo.gif, and you're good to go

Optional steps to tweak the logo section

  • Take note that the default size of the logo.gif is 298 pixels width and 63 pixels in height. To change the CSS code of the logo section, you will need to edit the style.css file
  • Open up style.css, under "General Style" locate the block of code like what we have above
  • Here you can edit the height of the main header as well as the width and height of the default logo dimensions

The navigation/menu of the template

File : All .html files

  • Change the text between the <a> tag to change a particular menu button for example About Us to Lear More
  • Change the property href="single.html" and link it to a desired webpage
  • Cut and paste class="current_page_item" and move it in a different <li> tag to specify a current page/link
  • Just copy or delete a single line if you want to add or remove a particular button

The featured slider (Image slider part)

File : index.html

  • You can edit the img src property to change the image
  • Images should not exceed 708 pixels in width and 367 pixels in height
  • Just duplicate the <li> block of code to add another featured item

The featured slider (Description slider part)

  • Edit the content of <h2> tag for the title of the featured item
  • you can edit the contents of <p> for the description of the featured item
  • just duplicate this block of code to add another featured item
